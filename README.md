# 校园失物招领系统简介 python486

## 项目概述

本系统是基于Python语言的校园失物招领平台，旨在帮助校园内的师生快速找回遗失物品，同时提供一个便捷的失物上报与认领体系。

## 技术栈

- 后端：Python + Django
- 前端：Vue
- 数据库：MySQL

## 系统角色

- 管理员：负责系统维护、数据管理与用户管理
- 用户：可上报失物、认领失物

## 系统特性
